# Web App (Next.js)

## Features included

- Admin login
- Dashboard summary cards
- Quick payment flow with `Mark as Paid`
- One-tap WhatsApp open buttons for tenant and owner
- Report export quick links (PDF/XLSX)
- PWA manifest scaffold

## Run

1. Set API base in env:
   - `NEXT_PUBLIC_API_BASE_URL=http://localhost:4000`
2. Install dependencies from repo root:
   - `npm install`
3. Start web app:
   - `npm run dev -w web`

## Routes

- `/login`
- `/dashboard`
